The Realizable, Responsive, Unobtrusive Unit (R2U2) is a stream-based runtime verification framework based on Mission-time Linear Temporal Logic (MLTL) designed to monitor safety- or mission-critical systems with constrained computational resources. R2U2 provides a python-based front-end compiler (C2PO) with deployable realizations in both C and embedded Rust.
